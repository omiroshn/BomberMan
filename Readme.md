# Bombeman

This project is a full implementation of a 3D Bomberman video game.

Frameworks and technologies used:
- C++
- CMake
- ImGui
- SDL
- ...


How to build:
```
git clone --recursive https://github.com/Liudok/Bombeman.git
cd Bombeman
mkdir build
cd build
cmake ..
make
```
(The execution file will be in the "bin" folder)
